UResource : 에셋/번들/리소스 파이프라인  

URefHandle : 참조 카운트를 추적하여, 0이 아닌 해제/미해제 종료를 에러로 검출.

ObjectPtr : 객체가 파괴될때 무효화 콜백을 전파하여, 댕글링 참조/NRE을 예방.

ObjectPool : 범용 오브젝트 풀 구현 ( Alloc/Free/Release 표준, 초기화/반납 훅 제공 ).

Uinterface : Layer기반 UI 매니저, UIController 수명주기 총괄, (UMediator) 브로드캐스트 지원

UMediator : 객체 간 메세지 중개 이벤트 버스

UUIController : 메인 뷰 (UUIViewBehaviour) 수명 주기 관리와 이벤트 라우팅

UUIViewBehaviour : 동적 생성되는 자식 ViewBehaviour 관리

SkillFactory : 어트리뷰트 기반 스킬 자동 등록 Factory 설계
